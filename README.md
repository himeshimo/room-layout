# Room Planner

部屋の寸法と家具サイズをもとに、  
「実際に置けるか」を視覚的に検証できる2Dレイアウトツールです。

家具の配置だけでなく、  
**はみ出し・重なり・開閉スペース**まで考慮してチェックできます。

---

## 🔗 Demo
https://himeshimo.github.io/room-layout/

---

## ✨ Features

- Room Zoneで部屋サイズをcm単位で作成
- 家具サイズをcm単位で配置
- カスタム家具の追加
- 家具同士の重なり検知
- はみ出し・ギリギリ判定
- 開閉スペース（クリアランス）表示
- ズーム / パン操作
- Undo / Redo

---

## 🚀 Usage

1. 縮尺を設定
2. Room Zoneで部屋を作成
3. 家具を追加
4. 重なり・はみ出し・開閉スペースを確認

---

## ⚠️ Caution

このツールは入力した寸法をもとにした概算です。  
実際の部屋や家具とは誤差が出る場合があります。

最終判断は必ず実測値で確認してください。
